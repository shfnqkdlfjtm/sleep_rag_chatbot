# Sleep Counselor (Spring AI, RAG, Voice Bridge)

## 요구 환경변수
- `OPENAI_API_KEY`
- `PINECONE_API_KEY`, `PINECONE_ENVIRONMENT`, `PINECONE_INDEX_NAME`
- `BRIDGE_BASE_URL` (파이썬 STT/TTS 브리지, 기본: `http://localhost:8001`)
